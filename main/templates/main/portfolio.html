Employee Information Management System

This full-stack web application was developed using PHP and MySQL to efficiently manage employee data, fiscal years, and tax slabs based on Pakistan’s 2025–2026 budget framework. The project focuses on real-world data handling, tax computation logic, and modular CRUD functionalities.

Project Features
  • Employee Information Module: Add, edit, delete employee records including personal and salary details. Data is displayed in a responsive table with inline editing support.
  • Fiscal Year Module: Manage fiscal years through dynamic forms. Integrated dropdowns allow users to select fiscal years linked with tax slab data.
  • Tax Slab Management: Create and manage tax slabs with salary ranges, tax percentages, and base tax values for each fiscal year.
  • Dynamic Tax Calculation: Automatically calculate annual and monthly tax based on salary and fiscal year with additional rules for high-income brackets.
Technology Stack

PHP (Core, using mysqli prepared statements), MySQL for backend and database, HTML and CSS for frontend structure and styling.

Geospatial Data Visualization Web Application

This full-stack web application was built using Python (Flask) and SQLite, integrating advanced geospatial data visualization features for interactive data analysis and mapping. The system supports dynamic CRUD operations, spatial data rendering, and embedded analytics through visual charts and maps.

Key Features
  • Complete CRUD: Create, edit, view, and delete geospatial records using Flask routes and SQLite.
  • Interactive Maps: Leaflet.js integration with multiple base layers including OpenStreetMap, Satellite, Topographic, and Dark modes.
  • GeoJSON Visualization: Display points, lines, and polygons with customized popups and thematic styling.
  • Real-Time Markers: Fetch markers from Flask endpoints dynamically and render them using custom icons.
  • Live Mouse Coordinates: Show coordinates in real time on the map interface.
  • Basemap Toggling: Enable users to switch between basemap styles with built-in Leaflet layer controls.
  • Analytics Dashboards: Plotly charts including bar and stacked bar charts rendered in Jinja2 templates for interactive data visualization.
Technology Stack

Python (Flask), SQLite for lightweight database management, Leaflet.js for geospatial interactivity, Plotly for data visualization, and HTML/CSS/Bootstrap for responsive UI.

Example Use Cases
  • Visualizing cities such as Islamabad, Lahore, Karachi, Murree, and New York using interactive markers
  • Displaying live GeoJSON datasets for spatial data analysis
  • Providing in-app map tools and dashboard-level analytics for location-based decision-making

This project demonstrates expertise in geospatial integration, RESTful architecture, client-side rendering, and full-stack development using Python and modern web technologies.

Secure SQL Query Interface with FastAPI & React

This full-stack web application allows authenticated users to submit natural language prompts, which are processed into SQL queries. It features secure authentication, modern UI, and strong separation between backend and frontend using FastAPI and React.

Key Features
  • Authentication System: Implemented secure JWT-based login with FastAPI and protected all sensitive endpoints.
  • Token Access Control: Utilized Depends, Form, and OAuth2PasswordBearer for token validation and secure form handling.
  • React Frontend: Developed a responsive UI using React, TypeScript, and Tailwind CSS. Includes a modern login page and query interface.
  • Session Management: Handled sessions using localStorage and React Router for navigation.
  • Cross-Origin Support: Enabled CORS middleware to support seamless development and deployment between client and server.
  • Error Handling: Implemented alerts for token expiry, invalid credentials, and server response errors.
  • SQL-Ready Backend: Configured local MySQL integration with clean API structure and routing.
Technology Stack

FastAPI (Python), JWT for authentication, React (with TypeScript), Tailwind CSS for styling, MySQL for backend data, React Router DOM for navigation.

Learning Outcomes
  • Strengthened backend security and token management practices with FastAPI
  • Improved frontend component handling and state management using React and TypeScript
  • Gained hands-on experience integrating secure full-stack login systems

This project showcases expertise in modern API security, UI/UX design, and full-stack integration of Python and JavaScript frameworks.
